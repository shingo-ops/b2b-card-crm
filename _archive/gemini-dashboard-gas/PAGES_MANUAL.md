| KPI | 計算方法 | 数値例 | 目標 |
|-----|---------|--------|------|

| This Month | 当月のDefectLog件数 | 3件 | 0件 |
| Defect Rate | 不具合日数 / 稼働日数 * 100 | 16.7% | 0% |
| Recurrences | is_recurrence=true の件数 | 2件 | 0件 |
| Recurrence Rate | 再発数 / 全不具合数 * 100 | 6.7% | 0% |

## 6. 共通コンポーネント

### Header

```
+------------------------------------------------------------------+
| [Title]                [Badge]        [Health]    [Refresh]       |
| 統合ダッシュボード      提案環境       Health: 85   [Refresh]      |
+------------------------------------------------------------------+
```

| 要素 | 説明 | 動的 |
|------|------|:----:|
| Title | アプリ名 | No |
| Badge | 環境識別（PROD/DEV/PROP） | No |
| Health | Health Score | Yes |
| Refresh | データ再読込ボタン | - |

### Tab Navigation

```
+------------------------------------------------------------------+
| [Overview] [Gemini] [Claude] [Defects] [GAS Projects]            |
|   active    normal   normal   normal      normal                  |
```
