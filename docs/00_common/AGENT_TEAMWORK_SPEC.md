| # | チーム名 | 目的 | 期待効果 |

| 1 | 商談支援（Golden Triangle） | 商談成功率最大化 | 成約率+35% |

### 3.2 チーム詳細

#### チーム1: 商談支援チーム
- **メンバー**: Ana + 心理学特化 + トップセールス + 伝え方特化
- **発動条件**: 商談前後の分析時

#### チーム2: 成長支援チーム
- **メンバー**: Buddy + 脳科学特化 + 認知科学特化 + コーチング特化
- **発動条件**: 成長段階の判定・介入設計時

#### チーム3: 体験設計チーム
- **メンバー**: デザイン特化 + 人間工学特化 + 認知科学特化 + 伝え方特化
- **発動条件**: UI/UX改善、メッセージング設計時

#### チーム4: 安定稼働チーム
- **メンバー**: システム開発特化 + 保守運用特化 + CTO + CISO
- **発動条件**: システム変更、障害対応時

#### チーム5: リスク管理チーム
- **メンバー**: 倫理特化 + 法務特化 + Devil's Advocate + 社内監査役 + 第3者監査役
- **発動条件**: 重要決定、コンプライアンス確認時

#### チーム6: 戦略立案チーム
- **メンバー**: CEO + CSO + CFO + CMO + 分析特化 + 顧客視点特化
- **発動条件**: 戦略レビュー、重大方針決定時

#### チーム7: 士気向上チーム
- **メンバー**: エンタメ特化 + ポジティブビジョン + メンタルヘルス特化 + CHRO
- **発動条件**: 定期イベント、士気低下兆候時
