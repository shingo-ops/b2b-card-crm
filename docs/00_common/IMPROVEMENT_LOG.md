# 改善ログ

## 概要
プロジェクト全体の改善履歴を記録するドキュメントです。

## 2025年1月

### 2025-01-11: GitHub連携開始
- **変更**: リポジトリ初期セットアップ
- **理由**: 開発ナレッジのバージョン管理とチーム共有
- **影響**: 全プロジェクト

### 2025-01-11: CRM SPA化
- **変更**: マルチページからSingle Page Applicationに変換
- **理由**: ページ遷移の高速化、操作性向上
- **影響**: CRMダッシュボード

### 2025-01-11: プルダウン設定のシート管理化
- **変更**: ハードコードから設定シート参照に変更
- **理由**: 選択肢の変更を容易に
- **影響**: CRM全体のプルダウン項目
